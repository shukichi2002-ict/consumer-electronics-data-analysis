# Consumer Electronics Data Analysis

##  Objective
Analyze consumer electronics sales data to identify trends, demand patterns, and business insights.

## Dataset 
- Source: Public sales dataset from GitHub  
- Author: Deepanshu Verma  
- Link: https://github.com/Deepanshu0154/Data-Analysis/tree/main/Sales%20Analysis

##  Tools
- Python (Pandas, Matplotlib)
- Jupyter Notebook

##  Key Findings
- Sales show strong growth with peaks in November and December (seasonal demand)
- Major cities like San Francisco, Los Angeles, and New York generate the highest sales
- Smartphones and accessories are top-selling products

##  Business Insight
Increasing demand for consumer electronics leads to higher demand for semiconductors (DRAM, NAND), which are essential for modern devices.
